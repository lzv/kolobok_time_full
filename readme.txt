Создатели кокоса выпустили следующую версию фреймворка, но изменения в нем могут потребовать доработки кода игры. Поэтому я выложил игру kolobok_time полностью, вместе с фреймворком cocos2D-x 3.2. 

Предупреждение: компиляция проверялась только под Linux (ubuntu), для других систем может потребоваться доработка.

Не забудьте установить зависимости!

----------------------------

kolobok time - простая 2d игра с элементами RPG, сделанная на фреймворке Cocos2d-x 3.2. 

Это еще не полноценная игра, а скорей заготовка для нее, основа. Для полноценной игры нужно добавить уровней, контента, да и нарисовать все покрасивей. Сделал же я эту игру для того, что бы получить опыт в самостоятельном выполнении проекта такой сложности. Если же кто-нибудь заинтересуется в развитии этой заготовки до полноценной игры, свяжитесь со мной: vic2_@mail.ru, сайт lzv.name.
